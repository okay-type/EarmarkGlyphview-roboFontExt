# Marky Mark Robofont Extension

by Jackson Showalter-Cavanaugh
www.okaytype.com

![Marky Mark Icon](/resources/ScreenShot.png?raw=true)

What does it this do? This extension adds a little triangle in the upper right corner of Robofont's glyph edit window showing the current glyph's markColor. 

In the future it would be neat to add a menu to change the current glyph's markColor from a dropdown list. But for now...
